Bootstrap Ölçülendirme ve Boyutlandırma

Resimleri Responsive Hale Getirme

Bootstrapde resimler için classlar mevcuttur
img-fluid class ı resmi responsive hale getirir
img-thumbnail class ı dışı çizgili küçük resim formatı verir

Diğer Boyutlandırma Yöntemi

w-50 : 50%
w-75 : 75%
w-100 : 100%
